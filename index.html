Swiper.js continuous autoplay smooth slider (Like marquee)

Post summary: Set Swiper.js move to work like marquee.

The autoplay of Swiper.js stops on each slide when moving. If you rather want to play like marquee and continue moving/sliding, then you just need to edit the CSS and Swiper config. Maybe it will be available in Swipers API in future, but currently (swiper 11.0.6) it is not.
